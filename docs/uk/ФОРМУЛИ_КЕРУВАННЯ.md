# Реєстр формул керування (UK)

## Конвенції

- Зовнішні сигнали: `PV`, `SP`, `OP`
- Внутрішні позначення: `y`, `r`, `u`
- Помилка: `e = r - y`

## Збурення по виходу

- `u_applied = clamp(u_ctrl + d_u(t), 0, 100)`
- STEP: `t >= DIST_START_S`
- PULSE: `DIST_START_S <= t < DIST_START_S + DIST_DURATION_S`

Код:
- `firmware/main.py::_disturbance_is_active`

## PID (паралельна форма)

- `u = clamp(P + I + D)`
- `P = Kp*e`
- `I[k] = I[k-1] + Ki*e*dt`
- `D = LPF(-Kd*d(PV)/dt)`

Код:
- `firmware/control.py::PIDParallelPercent.update`

## Релейне налаштування

- `A = PV_pp/2`
- `d_eff = (u_high - u_low)/2`
- `Ku = 4*d_eff/(pi*A)`
- `Pu = mean(period)`

Код:
- `firmware/identify.py::run_relay_tuning`

## Модельне налаштування (FOPDT)

- `ZN1_*` і `CC_*` з переходом у паралельну форму:
  - `Kp=Kc`, `Ki=Kc/Ti`, `Kd=Kc*Td`

Код:
- `firmware/identify.py::_model_rule_set`
- `firmware/identify.py::run_model_tuning`

## FOPDT ідентифікація

- Стаціонарність: `STEADY_WINDOW_S` і `STEADY_BAND_C`
- `K = (y_final - y_initial)/(u1 - u0)`
