# Стиль звітів (UK)

Область дії:
- `firmware/*.py`
- `runner/lab.py`

## Класи повідомлень

- `# PHASE:` початок фази
- `# INFO:` нейтральна інформація
- `# WARNING:` нефатальна проблема
- `# ERROR:` фатальна помилка, нагрівач вимкнено
- `# RESULT:` розрахований результат
- `# FINISH:` маркер завершення для синхронізації з host
- `# READY:` система в стані очікування

## Формат блоків

- Дозволений лише розділювач:
  - `# ========================================`

PV:xx.x SP:xx.x OP:xx.x
