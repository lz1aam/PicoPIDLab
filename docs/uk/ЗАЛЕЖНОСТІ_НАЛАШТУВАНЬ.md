# Залежності налаштувань (UK)

Документ описує допустимі комбінації параметрів у `firmware/profile.py`.

## CONTROL_MODE

- `ONOFF`
- `PID`
- `FUZZY`
- `MPC`

## PID

### PID_VARIANT

- `PID`, `2DOF`, `FF_PID`, `GAIN_SCHED`, `SMITH_PI`

### PID_ALGORITHM

- `PARALLEL` -> активні `KP`, `KI`, `KD`
- `IDEAL`/`SERIES` -> активні `KC`, `TI_S`, `TD_S`

### TUNING_RULE

Модельні:
- `ZN_1_P`, `ZN_1_PI`, `ZN_1_PID`, `CC_P`, `CC_PI`, `CC_PID`

Релейні:
- `ZN_2_P`, `ZN_2_PI`, `ZN_2_PID`, `TL_P`, `TL_PI`, `TL_PID`

## Телеметрія

- `INFO` -> тільки `# ...`
- `NORMAL` -> `PV/SP/OP`
- `MPC` -> `PV/SP/OP` + `YH/YP`
