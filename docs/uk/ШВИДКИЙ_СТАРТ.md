# Швидкий старт

Цільова версія: **v1.2.11**

Підтримувані сценарії:
- Thonny (прямий доступ до firmware)
- runner (`runner/lab.py`) для рецептів і збереження результатів

## 1. Підготовка

- Python 3.10+
- Thonny
- Пакети:

```bash
pip install pyserial pyyaml matplotlib numpy
```

- Скопіювати `firmware/*` на плату.

## 2. Thonny workflow

1. Запустити `firmware/main.py`.
2. Дочекатися `# READY:`.
3. Використовувати команди: `check`, `status`, `params`, `control|tune|model|monitor`, `stop`.

## 3. Runner workflow

```bash
python3 runner/lab.py
```

Команди host (`lab>`): `h`, `c`, `e <id>`, `s`, `k`, `u`, `x`, `b`, `q`.

Артефакти запусків:
- `runner/runs/<timestamp>__<EXPERIMENT>__<shortname>/`

## 4. Рекомендована послідовність

1. `model`
2. `tune`
3. `control`
