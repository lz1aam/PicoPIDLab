# Глосарій скорочень і технічних термінів (UK)

## Сигнали та величини

- `PV` — Process Variable, регульована величина (виміряна температура, °C)
- `SP` — Setpoint, задане значення
- `OP` — Output, керувальний вплив на нагрівач [%]
- `y` — внутрішній позначник виходу процесу (`PV`)
- `r` — внутрішній позначник завдання (`SP`)
- `u` — внутрішній позначник керувального впливу (`OP`)

## Структури керування

- `ON/OFF` — двопозиційне керування з гістерезисом
- `PID` — пропорційно-інтегрально-диференціальний регулятор
- `PI` — пропорційно-інтегральний регулятор
- `P` — пропорційний регулятор
- `2DOF` — PID з двома ступенями свободи
- `AW` — anti-windup
- `FF` — feedforward (випереджувальний канал)
- `MPC` — model predictive control
- `FUZZY` — нечітке керування (Sugeno)

## Моделювання й ідентифікація

- `FOPDT` — First-Order Plus Dead Time
- `K` — статичний коефіцієнт підсилення [°C/%]
- `tau` — стала часу [s]
- `theta` — запізнення [s]
- `SK` — Sundaresan-Krishnaswamy
- `BROIDA` — метод Бройди

## Правила налаштування

- `ZN` — Ziegler-Nichols
- `TL` — Tyreus-Luyben
- `CC` — Cohen-Coon
- `Ku` — граничне підсилення
- `Pu` — граничний період

## Форми параметризації

- `K`-форма: `Kp`, `Ki`, `Kd`
- `IDEAL`-форма: `Kc`, `Ti`, `Td`
- `Pb` — пропорційна смуга [%]
- `SPAN` — інженерний температурний діапазон [°C]

## Показники якості

- `TS_S` — період дискретизації [s]
- `RMSE` — середньоквадратична похибка
